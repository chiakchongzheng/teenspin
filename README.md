A fun **Progressive Web App (PWA)** to help teenagers discover their future careers through random occupation exploration. Built with modern web technologies.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎲 **Spin the Wheel** | Random career discovery with fun animations |
| 📖 **Job Overview** | Learn what each job actually does |
| 📅 **Daily Tasks** | See what a typical workday looks like |
| 🛠️ **Skills Needed** | Know what skills to develop |
| ✅ **Benefits & Challenges** | See the pros and cons of each career |
| 📚 **Free Resources** | Curated free courses, books, practice sites & communities |
| 📱 **PWA Ready** | Installable on mobile devices |
| 🎨 **Modern UI** | Built with shadcn/ui and Tailwind CSS |

Explore careers across **9 categories**:

| Category | Examples |
|----------|----------|
| 💻 Technology | Software Engineer, Data Scientist, Web Developer, Cybersecurity |
| 🎨 Design | UX Designer, Graphic Designer, Interior Designer, Animator |
| 🏥 Healthcare | Nurse, Doctor, Dentist, Veterinarian |
| 🍽️ Food | Chef, Restaurant Manager, Food Scientist |
| 💼 Business | Accountant, Marketing Manager, Entrepreneur |
| 📚 Education | Teacher, Professor, Tutor |
| ⚡ Trades | Electrician, Plumber, Mechanic, Carpenter |
| 👮 Public Service | Police Officer, Firefighter, Lawyer |

- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui + Radix UI
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/chiakchongzheng/teenspin.git

# Navigate to project
cd teenspin

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
